BIOMAUSSAN INTERNATIONAL — HOSTINGER READY LANDING PAGE

Archivos incluidos:
- index.html
- assets/css/styles.css
- assets/js/main.js
- assets/img/* (logo + 7 imágenes)

Cómo subir a Hostinger:
1. Entra al Administrador de archivos de Hostinger.
2. Abre la carpeta public_html del dominio.
3. Sube todo el contenido de esta carpeta respetando la estructura.
4. Verifica que index.html quede dentro de public_html.

Formulario:
- Actualmente el formulario muestra un mensaje visual de éxito.
- Si quieres conectarlo con un servicio real, puedes:
  a) usar FormSubmit / Formspree, o
  b) apuntarlo a tu backend.
- Edita el <form> en index.html para agregar action= y method= según tu servicio.

Botones CTA:
- "Shop in the U.S." actualmente baja al formulario de contacto.
- Puedes reemplazar esos href por el link real de tu tienda en EE.UU.

Tipografías:
- Usa Google Fonts (Inter). Si prefieres no depender de Google Fonts, puedo cambiarla a una fuente del sistema.
